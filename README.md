## 📌 Descripción general

Este portafolio reúne los proyectos desarrollados durante la materia de Desarrollo de Aplicaciones para Dispositivos Móviles.

En este trabajo se aplican conocimientos de programación, diseño de interfaces y manejo de datos mediante el desarrollo de aplicaciones funcionales.

###  Tecnologías utilizadas

- 🟦 Dart  
- 🟩 Flutter  
- 🧩 AppSheet  
- 🐙 GitHub  
- 💻 Visual Studio Code  
- 🎨 Material Design

- Programación en Dart

## 📱 Proyectos incluidos

### 📊 Proyecto 1: Análisis de Datos con Dart
**Descripción:** Aplicación de consola que permite leer archivos JSON y realizar búsquedas, filtros y análisis de datos.  

**Tecnologías implementadas:**
- Dart  
- JSON  
- dart:io  
- dart:convert  

---

### 📱 Proyecto 2: Generador de Palabras Aleatorias
**Descripción:** Aplicación móvil que genera palabras aleatorias y permite guardarlas en una lista de favoritos.  

**Tecnologías implementadas:**
- Flutter  
- Dart  
- Material Design  
- Provider
